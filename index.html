Rusticity

Make Rust Great Again

Rusticity is a modified Rust server with several quality of life mods and no pay2win, no exploiting/cheating tolerated, and an emphasis on fun.

View the map on playrust.io. Join our Discord and talk to your fellow players.

Our Server

The server features 3x XP / 3x gather, teleports, homes, clans, SIL, and many more features that enhance gaming while not diluting the excitement of combat and raiding. Rusticity aims to be the last great modified home for people who want a true Rust experience without the tedium of vanilla or the silliness of 10x arcade servers. Come play!

Can't find our server in the server browser? Press F1 in game and enter the following command to connect:

client.connect 23.229.84.86:27025

Our Staff

We have administrators in multiple timezones across the US and Europe, if you need help in game and nobody is about, your best bet is to create a ticket with the /ticket chat command.
