Our server is dedicated to running just Rust with an enterprise level connection and administrators located in many different timezones.

View the map on playrust.io.

Can't find our server in the server browser? Press F1 in game and enter the following command to connect:

client.connect 23.229.84.85:27025

Getting Help

To reach an admin please hop in game and use the /ticket chat command or Join our Discord and talk to your fellow players.
