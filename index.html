Rusticity

Make Rust Great Again

Rusticity is a modified Rust server with several quality of life plugins and no pay2win, no exploiting/cheating tolerated, and an emphasis on fun. The server features 3x XP / 3x gather, teleports, homes, clans, SIL, and many more features that enhance gaming while not diluting the excitement of combat and raiding. Rusticity aims to be the last great modified home for people who want a true Rust experience without the tedium of vanilla or the silliness of 10x arcade servers. Come play!

Our Server

Our server is dedicated to running just Rust with an enterprise level connection and administrators located in many different timezones.

View the map on playrust.io.

Can't find our server in the server browser? Press F1 in game and enter the following command to connect:

client.connect 23.229.84.86:27025

Getting Help

To reach an admin please hop in game and use the /ticket chat command or Join our Discord and talk to your fellow players.
